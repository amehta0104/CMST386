Describe how search engines work

Seatch engines work by "crawling" the internet using bots called spiders. These spiders follow links from page to page to find new content that has yet to be indexed. When using a search engine, relevant results to your search are pulled from the index and then displayed in an order determined by the algorithim that partiacular search engine is using

Explain the importance of keywords and descriptions as meta tags

The title tag amd the meta description tags should include keywords relevant to the content of the web page they describe. This helps Search Engines understand what the page is about and index the site accordingly

What constitutes a good heading hierarchy and include one example with more than one heading tag

A good heaing hierarchy is where you can clearly differentiate between titles ( and important content) to regular content. You also use heading hierarchy to show the reader what order to consume your content. For example, the title of the page should be an h1 tag. Once you use this h1, none of the other titles on the page should be an h1 tag. Also, when styling your content, good heading hierarchy will ensure that the sizes of the tags are all clearly different.

For example, an h2 tag may have a size of 20px, while and h3 or an h4 tag will then be a 10px size.
